# Delivery Zone Checker

A simple Spring Boot application to check if a set of coordinates is within a delivery zone. Built using **Java 17**, **Spring Boot 3.5**, and **Maven**.

---

## Features

- Check if a location (latitude, longitude) is within the delivery zone.
- Simple web interface using Spring MVC and Thymeleaf.
- Fully functional backend service with easy-to-update delivery zones.
- Ready for deployment or local testing.

---

## Technologies

- **Java 17**
- **Spring Boot 3.5**
- **Maven**
- **Spring MVC**
- **Thymeleaf**
- **HTML / CSS**
- **Bootstrap** (if you plan to add styling)
